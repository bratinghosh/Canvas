Running the Program:

1. Open the command prompt inside the Canvas folder.

2. Compile the application using the following command:
	javac CanvasApplication.java

3. Run the program using the following command:
	java CanvasApplication

Assumptions: 

1. The user can only draw either a vertical (x1 == x2) or a horizontal (y1 == y2) line.

2. The user has to enter the coordinates of upper left corner (x1, y2) and lower right corner (x2, y2) to print a rectangle.

3. The command entered by user should be of the following format:

[Command] [integer (if necessary)] [integer (if necessary)] [integer/character (if necessary)] [integer (if necessary)]

4. A canvas has to be created before the user starts drawing.
